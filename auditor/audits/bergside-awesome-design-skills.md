## Cross-Component
Two observations, neither PR-eligible (informational only):

1. **10 skills ship a body-empty `## Brand` section** — `## Brand` is immediately followed by `## Style Foundations` with no text in between, in: `artistic`, `bold`, `editorial`, `futuristic`, `geometric`, `minimal`, `neobrutalism`, `power`, `retro`, `square`. The repo's own README (`README.md` §"What is a Design Skill?" and §"Contributing") documents "Brand & mission — the design philosophy and visual identity" as expected `SKILL.md` content and asks contributors to "ensure your skill file includes all required sections: mission, brand, style foundations...". These 10 files satisfy every schema-level requirement (frontmatter, structure, markers) but leave the brand-narrative content blank, a documented-contract gap rather than a schema violation — hence no scoring penalty under the fixed rubric, but worth fixing for consistency with the other 57 skills (all of which have a populated `## Brand` paragraph).
2. **10 `registry-examples/*.png` marketing images have no corresponding skill** — `anime-marketing.png`, `candy-marketing.png`, `dark-marketing.png`, `forest-marketing.png`, `machine-marketing.png`, `mars-marketing.png`, `matcha-marketing.png`, `minimax-marketing.png`, `nostalgia-marketing.png`, `pragmatic-marketing.png` exist in `registry-examples/` but have no matching directory under `skills/`, no entry in `skills/index.json`, and no reference in `README.md`. These are orphaned assets — the README's own count ("67 design system skill files") and table are internally consistent and do not reference the extra 10 images, so this is unused-asset clutter rather than a broken link. Likely leftovers from a larger upstream `typeui.sh` registry; low confidence this is a defect versus an intentional staging area for skills not yet promoted to this repo.

`skills/index.json` itself has zero drift against disk: all 67 keys match all 67 `skills/*/SKILL.md` directories exactly, and `README.md`'s skill table references exactly the 67 `registry-examples/*-marketing.png` files that correspond to real skills (no broken image links).
